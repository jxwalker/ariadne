# Ariadne Brand

Ariadne is the product name for this control layer.

## Naming

- Product: Ariadne
- CLI command: `ariadne`
- Static operations surface: Ariadne Console
- Evaluation harness: Ariadne Trials
- Memory and sleep routines: Ariadne Loom
- Agent mail and state bus: Ariadne Thread

## Logo

The logo assets live in:

- `assets/ariadne-logo-square.png`
- `assets/ariadne-logo-lockup.png`
- `assets/ariadne-logo-simple.png`
- `assets/ariadne-mark.svg`
- `assets/ariadne-logo.svg`

Use `ariadne-logo-simple.png` for light documentation pages, `ariadne-logo-square.png` for app icons and compact surfaces, and `ariadne-logo-lockup.png` for wider brand treatments. The SVG files are lightweight fallback assets for code-generated surfaces.

The mark combines an `A`, a thread path, and a bounded control frame. It should be used where the system needs to signal source-grounded navigation through complex agent work.
