# decision-2026-05-15T22-38-53-515Z: Keep live adapters evidence-first

Status: accepted
Recorded: 2026-05-15T22:38:53.515Z

## Context

Roadmap adapters now cover NotebookLM, GSD2, worktree, Playwright, CI, CodeRabbit, infrastructure, and OpenScorpion surfaces.

## Decision

Keep adapters as file-contract and evidence-record producers until each live integration has an approved mutation or submission path.

## Consequences

- External systems are not mutated by default
- Humans can inspect artifacts before enabling live adapters
- Future work can add narrow approved live connectors
